C语言的类型定义：

```c
int 变量名 —— int型变量。
int 变量名[长度] —— int型数组。
int *指针名 —— 指向int型对象的指针。
int (*指针名)[长度] —— 指向长度为[]的数组的指针 —— 类型为：int (*)[长度]

```

* C语言的变量，是基本类型在前声明其数据类型，变量名的周围各种标志突出了这个变量的特色——数组[]、*指针、变量。

* C语言中，变量未经初始化是很危险的，它可能会存有残存的值。
