数组

```c
float a[10];               //0.声明一个可存放10个元素的float数组a
int a[5] = {1,2,3,4,5};

for(int i=0; i<10; i++){   //1.数组用for循环赋值
        scanf("%f", &a[i]);
    }
//不可以：
int a[5];
a = {1,2,3,4,5}; //原因有待补充...

for(int i=0; i<10; i++){   //2.数组输出也用循环，引用直接用a[i]
        printf("%f, ", a[i]);
    }
```
